RetailMart Data Pipeline is an end-to-end data engineering project built using Python, Pandas, NumPy, SQLite, and Streamlit. The project simulates a retail business workflow by extracting raw sales data, cleaning and transforming it, loading it into a database, and generating meaningful business insights.

Key features:

* Data ingestion from multiple CSV files
* Data cleaning and preprocessing
* Data transformation and merging using Pandas
* Revenue analysis using NumPy
* SQLite database integration
* SQL-based reporting and business insights
* End-to-end ETL (Extract, Transform, Load) workflow

Project workflow:
Extract Data → Clean Data → Transform Data → Load into Database → Generate Reports

Live Demo:
https://retailmart-data-pipeline.streamlit.app/

Technologies Used:

* Python
* Pandas
* NumPy
* SQLite
* SQLAlchemy
* Streamlit

This project demonstrates practical data engineering concepts and how raw retail data can be converted into actionable insights.
